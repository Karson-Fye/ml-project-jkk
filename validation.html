Feature Validation

Shows the correlation between two features being tied to the survivabilty of that passenger
Chi-Squared

We use the chi-squared test to validate features. It uses categorical data to measure if a statistical difference between the the features.

A chi-square p value of 0.5 or higher shows a lack of significance in the dataset.
Pearson Correlation Coefficiant (PCC)

We also use the PCC test to validate features. This test measures the relationship between a variable and target.

Since we are looking for any correlation, we turned all values positive for comparisons sake.

A PCC r value of 0.5 or lower shows a lack of significance in the dataset.

Discussion on Feature Validation

Both tests show that 'age', 'ticket', and 'sibsp' are less significant values.

Although both 'sibsp' and 'age' proved to be insignficant based off both tests, we decided to keep all features for our study.

All the features in use increase accuracy, even if only slightly. Since accuracy is our primary target for this experiment (and not speed), we keep these features.

This information served as a metric to show us what features proved to be determining factors for survival.
