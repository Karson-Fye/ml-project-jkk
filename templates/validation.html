Feature Validation
Heat Map:

A correlation heatmap shows how similar or different features are. Closely related features have a correlation value closer to 1 or -1, whereas dissimlar features will have values closer to 0.
Chi-Squared

We use the chi-squared test to validate features. It uses categorical data to measure if there is a statistical difference between the features.

A chi-square p value of 0.5 or higher shows a lack of significance in the dataset.
Pearson Correlation Coefficiant (PCC)

We also use the PCC test to validate features. This test measures the relationship between a variable and target.

Since we are looking for any correlation, we turned all values positive for comparisons sake.

A PCC r value of 0.1 or lower shows a lack of significance in the dataset.

Discussion on Feature Validation

Both our Chi-Squared and Pearson Correlation tests show that 'age', 'ticket', and 'sibsp' are less significant values.

Although 'sibsp' and 'age' proved to be insignficant based off both tests, we decided to only remove 'sibsp' since its removal created a slight increase in accuracy.
This left us with 8 features.
All remaining features increase accuracy, even if only slightly. Since accuracy is our primary target for this experiment (and not speed), we keep these features.
